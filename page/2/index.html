🌾Python 异步编程asyncio模块
2019-11-21

asyncio 模块最大特点就是，只存在一个线程，跟 JavaScript 一样。
由于只有一个线程，就不可能多个任务同时运行。asyncio 是"多任务合作"模式(cooperative multitasking)，允许异步任务交出执行权给其他任务，等到其他任务完成，再收回执行权继续往下执行，这跟 JavaScript 也是一样的。
