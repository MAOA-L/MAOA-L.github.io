Supervisor

2020-03-09

  1. 安装 建立软连接

    yum install supervisor

  2. supervisor 本身

项目 /etc/supervisord.conf 配置了 supervisor的基本配置文件，
取消[inet_http_server] 配置单元的注释
使用 supervisord -c /etc/supervisord.conf
或者supervisorctl -c /etc/supervisord.conf 启动supervisor，修改完配置文件之后，使用supervisorctl reload重启supervisor
