Nginx

简介记录nginx的安装

需要的依赖环境

yum -y install pcre pcre-devel
yum install openssl openssl-devel

编译 && 安装

./configure --prefix=/usr/local/nginx --with-http_ssl_module
make && make install

启动

nginx

重启

nginx -s reload

指定配置文件的启动

nginx -c /usr/local/nginx/conf/nginx.conf
